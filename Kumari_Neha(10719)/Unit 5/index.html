Initialize Git Repository & Deploy Website

Aim

To initialize a Git repository, commit changes, push the project to GitHub, and deploy a personal portfolio website using GitHub Pages or Netlify.

Tools Used

  • HTML & CSS
  • Git
  • GitHub
  • Netlify / GitHub Pages

Steps Performed

  1. Created a simple HTML portfolio website.
  2. Initialized Git repository using git init.
  3. Added files using git add ..
  4. Committed changes using git commit -m "Initial commit".
  5. Pushed the repository to GitHub.
  6. Deployed the website using GitHub Pages / Netlify.
